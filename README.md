# Trading Bot

Торговый бот для Bybit с трёхэкранной стратегией.

## Возможности
- Анализ тренда (D1)
- Поиск зон входа по Фибоначчи (H4)
- Генерация сигналов (WATCH/LIMIT/INSTANT)
- Два режима: PAPER и LIVE
- Управление позициями и статистика

## Структура
- `main.py` — точка входа
- `core/` — основные модули
- `monitor_three_screen.py` — монитор
- `config/` — конфигурация (не в Git)
